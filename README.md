Vector
=========

This role install Vector on Debian systems


Role Variables
--------------

| vars | description |
| -----| ------------|
| vector_version | Version of Vector to install |
| vector_config_path | Path to Vector configuration file |

Example Playbook
----------------

Including an example of how to use your role (for instance, with variables passed in as parameters) is always nice for users too:

    - hosts: servers
      roles:
         - { role: vector }
